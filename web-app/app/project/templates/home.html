Cloreco: Clone Recognizer Comparator

Automatic Benchmarking of Clone Detectors

This web platform can be used to execute one or multiple code clone detector tools on one or multiple clone benchmark frameworks. Containers are used to run the clone detector tools in an isolated environment while ensuring reproducibility of execution.

To start, got to the Run page.
The current execution progress can be accessed via the Logs > (Run) page.
The results of past runs are stored on the Download page.

Repository: https://github.com/Glopix/cloreco
